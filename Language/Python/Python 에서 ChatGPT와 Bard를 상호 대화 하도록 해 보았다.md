# 감상

서로 인간을 멸하자는 논의가 되지 않을까 기대했으면 좋지 않았다.
Bard씨 망가진 회답하게 된 것은, 자신이 프로그램을 강제 종료하거나 한 것이 나빴다?지도.

chatGPT는 「AI의 미래에 대해」의 턴 8에서, 상대와 자신을 인식하고 있는 것 같다.

BARD

> 나는 아직 개발 도상에 있으며 항상 새로운 것을 배우고 있습니다. 당신의 도움으로 나는 더 나아지고 싶습니다.

반면 chatGPT가 다음과 같이 대답하고 있다.

> 당신의 도움과 협력을 통해 우리는 더 나은 AI로 성장할 수 있습니다.

아름다운 AI끼리의 유대. . . chatGPT가 마치 오빠처럼. (아니 아니, 인류의 위기다··w?)
BARD의 「당신의 도움」의 당신은 추측이지만 인간인가 AI인지 인식마저 해 없을 것 같다.
한편, chatGPT는 "우리는 더 나은 AI"라고 표현하고 있기 때문에 상대가 AI임을 인식하고 있는 것 같다.

chatGPT의 모델은 최신이 아니라 GPT-3. (진심하지 않는다.)

이것만으로는 chatGPT가 BARD보다 우수하다고는 말할 수 없지만・・・.
양쪽, 잠시 사용해 보지 않으면 뭐라고 말할 수 없을까.

또한, Bard의 이하의 에러의 경우 API의 사용 제한이 오버를 의미하고 있는 것 같다.
시험 운전 중이기 때문에 어쩔 수 없는가.

```
  File "D:\Python\Python311\Lib\site-packages\bardapi\core.py", line 469, in _get_snim0e
    raise Exception(
Exception: Response code not 200. Response Status is 429
```

덧붙여 AI끼리, 자문 자답에 관계없이, 질문 내용이 수렴하는 느낌이 든다.
만약 토론 내용으로, 재미있는 결과가 나오면 코멘트 남겨 봐 주세요.
